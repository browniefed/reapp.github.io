An easy way to build apps

Reapp is a set of modules built on node and JavaScript that lets you build cross-platform mobile apps in harmony.

Sign up to the mailing list to get early updates and information on development:

Interactive, additive animations UI Framework CLI with build tools

npm install -g reapp reapp new myproject

React

It's all Javascript, all the way down. From html, to style, to logic, everything is based on small, composable components.

UI Kit

Built on a UI kit that's lightweight, but supports themes, custom animations, and all the elements you've come to expect.

Structured

Includes a completely customizable stack to start you off. With immutable data, caching, routing, and the build system all figured out.

Composable

The entire stack is optional. Pick and choose pieces you want, and realize smaller bundles and build times when you do.
