A modern mobile app platform

re/app is a set of modules built powered by npm and React that brings harmong to building cross-platform apps.

Sign on to the mailing list to know about important releases:

UI Framework CLI with build tools Helpful modules

Standard

Powered by React and fully component-ized. Everything is JavaScript, even the styles.

Powerful

Our UI kit is lightweight and modular, supports themes, interactive animations.

Helpful

A CLI that generates a well-designed React app. Immutable data, caching, routing, and the build system.

Composable

Pick and choose pieces you want, realize smaller bundles and build times when you do.

npm install -g reapp reapp new myproject
